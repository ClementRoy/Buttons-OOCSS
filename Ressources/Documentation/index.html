- Changing color - top

There are 11 colors available.
To specify a color, you just need to add the name of the color as class to your buttonPro element.
You don't need to specify the white one, because that is the default.

These are the colors (class-names) available:

  1. light_blue
  2. blue
  3. green
  4. yellow
  5. orange
  6. red
  7. pink
  8. purple
  9. grey
  10. black

- Other features - top

You can apply the buttonPro class to any element, not just links.
You may especially be interested in adding the buttonPro style to inputs or buttons.

Examples:



You can also insert icons inside the buttons.
All you need to do is find a nice image and insert it inside the link, using the <img /> element, as you would usually do in html.
For a better result, the height of the image should be the same as the font-size of the text of the button (12px for the medium sized button, 11px for the small one, 20px for the large one).

C) CSS - top

The buttonPro.css file is divided into sections using comments.

  1. Defaults & General Styling:
    Here is the defined the basic buttonPro class, the three sizes available and the optional rounded version. Here is also defined the appearance of the default (white) button.
    If you want to make changes to the font, padding or if you want to add margins, this is the right place to look.
  2. Color Variations:
    Here is defined the appearance of each color variation. Every color has different styling for the normal, hover and active state.
  3. Extras:
    This is were the google, twitter, facebook and wordpress button designs are recreated. They don't have anything to do with the buttonPro class. Each of these buttons has its own custom design, custom font, size etc.
    If you are not interested in any of these button designs, you can easily get rid of this part of the code.
